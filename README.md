# Rasheed Pharmacy - Frontend

Modern, responsive web application for Rasheed Pharmacy customers and staff.

## Tech Stack
- React & Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- Vanilla CSS (Custom Design System)

## Setup instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Environment Setup:**
   - Copy `.env.example` to `.env`
   - Fill in `VITE_API_URL` (usually `http://localhost:5000/api`)

3. **Run Locally:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```

## Development Features
- **Auto-fill Forms:** Logged-in users get their details automatically filled.
- **Real-time Tracking:** Staggered animations and live status updates.
- **Owner Dashboard:** Secure access to inventory and prescriptions.
